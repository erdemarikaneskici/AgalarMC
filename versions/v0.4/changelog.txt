# v0.4 Changelog
**New Mods**

- Added "Punchy" mod. (It does not support some items, so for the time being those items will use the vanilla animations.[Extendo-grip, Plunger Launcher, Linked Controller etc. Other than that it is one hundred percent compatible.)

- Added "Spear Backport" mod. (Alongside "Copper Age Backport" mod, this mod acts as a placeholder until "Vanilla Backport" mod adds theese features natively.)
