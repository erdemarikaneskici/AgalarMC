# v0.4 Changelog

## 🇬🇧 English

**New Mods**

- Added "Punchy" mod. (It does not support some items, so for the time being those items will use the vanilla animations. Extendo-grip, Plunger Launcher, Linked Controller etc. Other than that it is one hundred percent compatible.)
- Added "Copper Age Backport" mod.
- Added "Spear Backport" mod. (Alongside "Copper Age Backport" mod, this mod acts as a placeholder until "Vanilla Backport" mod adds these features natively.)

**New Resource Packs**

- Added "Drigo 3D Lanterns x Punchy" resource pack.
- Added "HyperPunchy" resource pack. (Some vanilla "Punchy!" animations are too wobbly for my taste, "HyperPunchy" fits better.)
- Added "Modded Swords x Punchy" resource pack. (For the wrench in the "Create" mod.)

**Updated**

- Updated "MakeUp Ultra Fast" shader.
- Updated existing mods.

## 🇹🇷 Türkçe

**Güncellenenler**
