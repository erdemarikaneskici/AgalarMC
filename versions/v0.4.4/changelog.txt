# v0.4.4 Changelog

## **Added**
- **"BetterPingDisplay"** (Displays the exact ping of other players on the Tab screen)
- **"ChatImage"** (Lets images/GIFs render in chat)
- **"Explosive Enhancement"** (Changes the explosion effect to a better yet vanilla fitting one)
- **"NeoLanguageReload"** (Better language search and selection, and extra features like multilingual item search)
- **"SearchStats"** (Lets you search spesific stats in the statistics screen)
- **"Cubes Without Borders"** (_**Disabled by default**_)

## **Removed**
- **"DarkWindowBar"** (_**Disabled by default.**_ It causes the game to not being able to boot up in fullscreen)
- **"Crash Assistant"** (_**Disabled by default,**_ enable if you are facing issues after tweaking the pack. It is not a necessary mod if you just use the provided pack as is.)
